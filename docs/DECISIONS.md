لا نكتب فيه الأكواد، بل نسجل القرارات الهندسية التي نتخذها، مثل:

لماذا أنشأنا مجلد assets؟
لماذا فصلنا ملفات CSS وJavaScript؟
لماذا اخترنا JSON بدل قاعدة بيانات؟
لماذا جعلنا الشهادة "إفادة" وليست "شهادة رسمية"؟
لماذا اخترنا GitHub Pages للاستضافة؟

هذه القرارات غالبًا تُنسى مع مرور الوقت، لكن توثيقها يجعل المشروع أكثر نضجًا ويسهل أي تطوير أو انتقال لفريق آخر في المستقبل.
